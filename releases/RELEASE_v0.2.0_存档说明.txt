ALUCOLUX® Cost Calculator — 版本存档说明
========================================

定稿版本：v0.2.0
存档日期：2026-04-18

本文件夹用于存放「发布用」源码包；同项目在仓库根目录另有 Git 标签 v0.2.0 与本内容对应。

包含内容（以实际压缩包为准）：
- 完整 Python 源码（app.py、core/、docs/ 等）
- requirements.txt、default_config.json、color_cost_db.csv 等运行所需文件
- 用户手册生成脚本：docs/generate_alucolux_user_manual_rtf.py
  （运行后可在项目根目录生成 ALUCOLUX_用户手册.rtf）

不包含 / 说明：
- 不含 Python 解释器与虚拟环境；目标机器需自行安装 Python 3 并通过 pip 安装依赖。
- saved_calculations/ 下示例 JSON 若存在，为开发测试数据，正式部署可清空。
- 若 ALUCOLUX_用户手册.rtf 体积异常偏大（例如上百 MB），多为用 Word 反复另存产生冗余；可删除后执行上述脚本重新生成（正常应在约几十 KB 量级）。
