# ALUCOLUX® 成本计算小程序功能详细描述（Cursor开发规格）

## 1. 总体目标
开发一个**轻量、灵活、可交互**的小程序，核心是**读取File 1的变量表**，让用户随时修改所有参数。

## 2. 核心功能需求（重点更新）

### 功能1：参数输入界面（全部变量可调）
- 自动加载《ALUCOLUX_成本计算方法精确描述.md》中的“可调核心变量”表格
- 提供输入框/下拉菜单，让用户修改**每一个变量**的默认值
- 支持“保存为配置文件”与“加载配置文件”

### 功能2：一键计算
- 严格执行File 1中“精确计算步骤”的**变量公式**
- 所有计算必须使用File 1定义的变量名（不能写死数值）

### 功能3：输出格式（严格按此风格）
输出必须是**Markdown格式**，完全模仿我们之前的“超级简单版”对话：
- 标题 + 本次订单参数
- 六个清晰步骤（每步白话解释 + 公式（使用变量名） + 精确数字）
- 第五步给出总直接成本 + 单位保本价 + USD价格

### 功能4：技术要求
- Python + Streamlit（推荐）
- 把File 1的内容解析为Python字典（变量名 → 默认值）
- 所有公式直接引用变量字典
